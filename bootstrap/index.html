GET TO KNOW

Overview

Bootstrap is the most popular framework made to build new projects - responsive, mobile-first websites. This framework is open source.

What’ more, Bootstrap is completely free to download and use.

Grid System

Bootstrap’s grid system uses a series of containers, rows, and columns to layout and align content. It’s built with flexbox and is fully responsive.

Components

Bootsrap has an extensive collection of components, for instance, navs, buttons, collapse and so on. They are already styled in advance.

GET TO START

Bootstrap is very powerful and straight forward. By simply adding corresponding classes onto your html file you can easily take advantages from responsive grid system, extensive prebuilt components and powerful plugins built on jQuery that are provided by Bootstrap.

Two most common ways to include Bootstrap in your project: You can either download the documents or do it through the Content Delivery Network (CDN). Using BootstrapCDN can help you to add Bootstrap quickly.
